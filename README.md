# EchoSoul - Personal AI Companion

EchoSoul is a personal AI companion that remembers you, grows emotionally and behaviorally over time, preserves memories and personality, mirrors emotions, and becomes a "digital extension" of yourself.

## Features

### Core Features
- **Persistent Memory System**: Remembers everything about you
- **Adaptive Personality**: Evolves based on your interactions
- **Emotion Recognition**: Detects and responds to your emotions
- **Life Timeline**: Chronological view of your memories
- **Private Memory Vault**: Encrypted storage for sensitive memories
- **Legacy Mode**: Preserve your digital self for the future

### Streamlit Features
- Interactive chat interface
- Emotional timeline visualization
- Memory vault with encryption
- Personality customization
- Conversation analytics
